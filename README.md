Netflix-Clone is a web application that replicates the core features of the popular streaming platform, Netflix. The application provides users with a selection of movies fetched from the TMDB API, categorized into sections such as "Now Playing", "Upcoming", "Popular", and "Top Rated". The main page showcases a featured movie with its title, release date, and a brief overview. Users can browse through the movie listings, which are presented in horizontally scrollable rows. Each movie card offers a like/save feature, allowing users to save their favorite movies. The application also integrates with Firebase for authentication, enabling users to sign up, log in, and manage their account. The navigation bar at the top provides quick access to the main page, user account, and authentication features.
